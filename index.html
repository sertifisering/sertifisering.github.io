Korea Register Vessel Risk Analysis & Tracking System

A real-time monitoring platform where administrators or vessel accounts can view and customize vessel risk dashboards. The dashboard displays adjustable risk graphs, while the navigation page provides live vessel data including total risk indicators, key parameters, and position tracking. The resource section allows users to view or replace Bayesian network models used in analysis. I handled the full-stack system design, development, and deployment in collaboration with the algorithm team.

Subsea Cable Installation and Stabilization System

A system for planning subsea cable installation routes through an interactive map interface. Users can select a target area, load project data, and run deep learning algorithms to calculate optimal paths. Results appear in an Excel-like spreadsheet viewer for visualization and editing, allowing recalculation based on updated parameters. The system also visualizes seabed and installation data by depth levels to identify suitable zones. I designed, developed, and deployed the entire system and collaborated with the algorithm team.
